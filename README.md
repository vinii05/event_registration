# Online Event Registration Portal

HTML5 experiment demonstrating media elements (image, audio, video) and form inputs (text, email, date, radio, select, textarea, file upload).

## Features
- Event banner, promotional video, and background audio.
- Registration form with HTML5 validation.
- Responsive layout tested for different screen sizes.

## How to Run
1. Clone or download this repository.
2. Open `event.html` in any modern web browser.
