# Online Event Registration Portal (HTML5)

This project is an Experimentâ€‘1 implementation for an online event registration promotional portal using HTML5. It demonstrates media elements (image, audio, video) and interactive form elements with basic validation.

## Features

- Event banner image with alternate text.
- Promotional video with playback controls.
- Background/intro audio with controls.
- Registration form with:
  - Text, email, and date inputs
  - Radio buttons and dropdown selection
  - Textarea for additional comments
  - File upload for attachments
- Basic responsive layout tested by resizing the browser window.

## How to Run

1. Download or clone this repository.
2. Ensure all media files (e.g., `banner.jpg`, `promo.mp4`, `intro.mp3`) are in the same directory as `event.html`.
3. Open `event.html` in any modern web browser (Chrome, Edge, Firefox, etc.).
